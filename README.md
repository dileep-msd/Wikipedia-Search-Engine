# Wikipedia-Search-Engine
### Instructions for running
* **Index creation**
```
bash index.sh <path_to_dump> <index folder>
```
* **Searching**
```
python3 search.py
```
* **Query Format**
  * **Normal query** - Enter words
  * **Field query** - title:TITLE category:CATEGORY infobox:INFO ref:REFERENCE body:BODY
